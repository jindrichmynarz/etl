The {{page.title}} component provides a form to fill in DCAT-AP v1.1 Distribution metadata. The typical usage is in each data preparation pipeline to store and produce distribution related metadata. It can use the output of the DCAT-AP Dataset component to reuse the Dataset IRI, languages and issued and modified dates. Its output can be used by the DCAT-AP to CKAN component for loading the metadata into CKAN and enriched by the VoID Dataset component.
