The {{page.title}} component queries a remote SPARQL endpoint using a SPARQL SELECT query, using the scrollable cursor technique for OpenLink Virtuoso. This is helpful for getting larger results from an OpenLink Virtuoso SPARQL endpoint which would not return such a large result normally. This component is often used in combination with the SPARQL endpoint chunked component.
