--- short: t-rdftofile type: transformer color: indigo title: RDF to File description: Converts RDF data to an RDF file. ---

Transformer, allows the user to convert RDF data (RDF data unit) to an RDF file (Files Data Unit).

Output file name
File name under which the file will be visible in the pipeline, including extension.
Output format
RDF serialization of the output file.
IRI of the output graph
IRI of the only named graph that will be created from the input triples. Valid only for the RDF Quad formats.

The {{page.title}} component takes the RDF data on the input and serializes it into a file in the chosen RDF serialization. The component also supports RDF quad serializations. However, so far, LP-ETL only supports RDF triples. Therefore, to produce a quad file, the IRI of the output graph has to be specified and all the input triples will be put into this graph.
