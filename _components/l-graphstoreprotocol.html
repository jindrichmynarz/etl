The {{page.title}} component takes the input RDF data and loads it to a remote SPARQL endpoint using the SPARQL 1.1 Graph Store HTTP Protocol. Unfortunately, the implementation of this standard varies throughtout triplestores, this is why you need to choose the target implementation and why this component will probably not work with any other implememtation.
