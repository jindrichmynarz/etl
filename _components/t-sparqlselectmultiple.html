The {{page.title}} component creates CSV files using SPARQL SELECT queries based on the input RDF data. It can only be configured at runtime using RDF configuration, which can be generated by another component.

Runtime configuration

Below you can see sample runtime configuration for the component. You can also import a sample pipeline. Note that RDF blank nodes are not allowed in runtime configurations.

{% highlight turtle %} @prefix ss: . a ss:Configuration ; ss:query "SELECT ?s ?p ?o WHERE { ?s ?p ?o}" ; ss:fileName "table1.csv" . a ss:Configuration ; ss:query "SELECT ?s ?p ?o WHERE { ?s ?p ?o} LIMIT 1" ; ss:fileName "table2.csv" . {% endhighlight %}
